# Dmoney & Booking API Performance Testing (JMeter)

This project contains **Performance & Load Testing Collections** using **Apache JMeter** for:
-  **Dmoney Transaction System**
-  **Booking API System**

The goal is to **evaluate server performance** under different load conditions and ensure system stability for real-time usage.

---

##  Project Objectives
-  Measure server response time & throughput  
-  Identify performance bottlenecks  
-  Evaluate stability under increasing load  
-  Analyze resource behavior under stress  
-  Ensure reliability of core financial & booking operations  

---



##  Technologies Used
-  **JMeter**
-  **Postman**
-  **Node.js**
-  **GitHub**
-  **Excel (Result Analysis)**

---

##  How to Run
- Clone this project
- Create collection
- Setup pre requisite on jmeter
- Run this entire collection
- Follow up the summary and table to see output

---

## Prerequisite :
- Install Jmeter
- Setup Home environemt
- Install gitbash
